## Environment Variables

Set the environment variables in your shell:

```bash
AWS_ACCESS_KEY_ID="YOUR_ACCESS_KEY_ID"
AWS_SECRET_ACCESS_KEY="YOUR_SECRET_ACCESS_KEY"
```

Execute the script using the following command:
```bash
go run main.go
```
